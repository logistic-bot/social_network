# social_network
Python social network kata
https://kata-log.rocks/social-network-kata

------------------------

# Install & Use

```bash
git clone https://github.com/logistic-bot/social_network.git
cd social_network
mkdir data
mkdir messages
python main.py
```

## Requierments

- Python 3

# Commands

`rg` : Read global messages
`pg` : Post a global message
`lu` : List users
